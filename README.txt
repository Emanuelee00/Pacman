PACMAN
======

CONTROLS
--------
Arrow keys   Move Pacman
ESC          Pause / Back to menu

CONFIGURATION
-------------
A config.json file is included alongside the executable.
Edit it to customize: lives, seed, level sizes, ghost strategies (euclidean / bfs).

OPTIONS
-------
--config <file>    Load a custom configuration file
                   Example: ./pacman --config myconfig.json

CHEAT CODES
-----------
torretto    Extra life
42          Toggle wall visibility
level       Jump to level (type "level" then a digit 1-9)
false       Disable active cheat
